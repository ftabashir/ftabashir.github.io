# Farzad Tabashir — Portfolio & Resume Website

The source code for my personal portfolio and professional resume website.

## Live Website

🔗 **[ftabashir.github.io](https://ftabashir.github.io/)**

---

## Features

- **Responsive Design**: Built with a clean two-column grid on desktop that gracefully collapses to a single-column layout on mobile devices.
- **Dark & Light Themes**: Sleek togglable interface (persisted via `localStorage`) adjusting variables, glowing accents, and backdrop filters.
- **Interactive Skills Filter**: Interactive badges allow visitors to highlight specific roles and experience details matching their selected skill.
- **Print Optimization**: Tailored `@media print` style blocks formatting the page for clean, high-contrast print outputs and PDF exports.
- **Semantic HTML**: Fully accessible layout structured with modern HTML5 markers (`<header>`, `<main>`, `<section>`, `<article>`, `<aside>`).
- **Zero Heavy Dependencies**: Written in clean, vanilla HTML, CSS, and JavaScript.
